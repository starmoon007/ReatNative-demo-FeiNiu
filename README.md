# ReatNative-demo-FeiNiu

### 使用ReactNative 高仿的飞牛网电商工程
1. 项目基本架构

   AppDelegate: 项目入口
   
   TabController: 标签栏控制器，拖拽5个TabNavigatior
   
   TabNavigatior: 承载具体业务页面 其中自定义了导航栏组件--NavigationBar
